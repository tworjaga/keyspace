### Contribution Types

This project follows the [all-contributors](https://allcontributors.org) specification. Contributions of any kind are welcome!

- Code
- Documentation
- Design
- Ideas & Planning
- Bug Reports
- Tests
- Tools
- Infrastructure
- Packaging
- Translation
- Security


## How to Contribute

We welcome contributions from everyone! Please see our [Contributing Guide](CONTRIBUTING.md) for details on how to get started.

### Ways to Contribute

1. **Code**: Fix bugs, add features, improve performance
2. **Documentation**: Fix typos, add examples, improve clarity
3. **Testing**: Write tests, report bugs, verify fixes
4. **Design**: Improve UI/UX, create graphics, design icons
5. **Translation**: Translate documentation and UI
6. **Community**: Help others, write tutorials, spread the word

## Recognition

## Become a Contributor

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request

See [CONTRIBUTING.md](CONTRIBUTING.md) for detailed instructions.
